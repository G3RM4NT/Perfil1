VexSoluciones

Acerca de Vex Soluciones Somos una firma tecnológica especializada en el diseño y programación de aplicaciones web, aplicaciones móviles, marketing digital, ecommerce y e-learning con fuerte presencia internacional. Nuestro equipo de expertos usará todo el conocimiento de nuestra red a favor de su negocio.

Premper Premper

Somos apasionados en nuestro trabajo, constantemente nos reinventamos, implementando tecnología nueva siendo Ruby on Rails nuestro lenguaje de programación principal, pero también usamos Bootstrap / jQuery Mobile / CSS y html5 y para aplicaciones para teléfonos usamos React Native. Toda la empresa utiliza tecnología de código abierto (Ubuntu y otros sistemas operativos). Nosotros creemos en la importancia de la libertad de información y el uso de software libre (es decir Linux y software de código abierto) es una parte importante de nuestra filosofía empresarial. Apoyamos activamente la Free Software Foundation.
